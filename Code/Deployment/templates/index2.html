ABOUT





CORNEA is a web application developed by three Data Science graduate students of Illinois Institute of Technology - Olugbenga Abdulai, Kinjal Kachi, and Sujay Puvvadi. The COVID-19 Recognizer tool is a machine learning model for X-Ray images that classifies an image as one of three classes - COVID-19, Viral Pneumonia, or Normal. Radiologists might find this tool useful for disease recognition.
